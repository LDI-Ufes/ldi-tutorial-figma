1. Ferramentas

O Figma tem ferramentas bem parecidas com os programas mais utilizados do mundo do design como por exemplo o pacote Adobe.

1.1 Move

Ferramenta padrão do mouse. Permite: seleção, movimentação, transformação livre de todos os elementos dentro de um documento.
tecla [v]

1.2 Scale

Permite a transformação de elementos em escala, mantendo a proporção do mesmo.
tecla [K]

1.3 Frame

Criação de quadros (relativos às pranchetas de programas externos de design), possui predefinições de telas de dispositivos móveis como: desktop, tablets, celulares e etc.
tecla [F]

1.4 Slice

A ferramenta fatia serve para capturar uma área de um quadro e exportar de forma ágil e fácil, sem precisar unir elementos em grupo.
tecla [S]

1.5 Formas

Permite a criação de formas.
retangulo [R]linha [L]seta [SHIFT] + [L]elipse [O]imagem [CTRL] + [SHIFT] + [K]

1.6 Pen

A caneta permite a criação livre de pontos de linha e de curva. Ferramenta convencional de programas de design, possui as mesmas funcionalidades.
tecla [P]

1.7 Pencil

O lápis permite a criação livre de desenhos. Ferramenta convencional de programas de design, possui as mesmas funcionalidades.
teclas [SHIFT] + [P]

1.8 Texto

Inserção de texto e manipulação de suas propriedades tipográficas como: espaçamento entrelinhas, espessura, tamanho, tipos de fonte, etc.
tecla [T]

1.9 Hand

Permite o controle de visão da área do documento. Ferramenta convencional de programas de design, possui as mesmas funcionalidades.
tecla [H] ou [BARRA DE ESPAÇO]

1.10 Comments

Permite a inserção de comentários dentro de um documento. Bastante eficaz para possíveis feedbacks em equipe.
tecla [C]

2.1 Copiar / Recortar

Cópia ou recorte de um elemento, permitindo a cola do mesmo elemento.
copiar [CTRL] + [C]recortar [CTRL] + [X]colar [CTRL] + [V]

2.2 Restringir Mov.

Restringe a movimentação de um elemento em X e Y.
tecla [SHIFT]

2.3 Duplicar

Duplica uma ação realizada anteriormente. Ex: Cópia/Cola e mesma margem.
teclas [CTRL] + [D]

2.4 Alinhamentos

O Figma possui comandos interessantes para o alinhamento de elementos
↑ [ALT] + [W]↓ [ALT] + [S]→ [ALT] + [D]← [ALT] + [A]HORIZ. [ALT] + [H]VERT. [ALT] + [V]

2.5 Opacidade

Para alterar a opacidade de quaisquer elementos utilizamos as teclas númericas diretamente.VERT. [ALT] + [V]

2.6 Zoom

O controle da área de tela do documento é facilitado através de comandos simples.
TECLAS NUM. [1] à [9] - 10% ~ 90%TECLA NUM. [0] - 100%TECLAS [NUM] + [NUM] - 01% ~ 99%

3. No Documento

Em seu documento o Figma possui recursos e interações bastante práticas que facilitam o seu fluxo de trabalho.

3.1 Importação e Exportação

O Figma possui muitos recursos de exportação e importação de elementos. Abaixo apresentaremos alguns recursos que utilizamos. IMPORTAR IMAGEM [CTRL] + [SHIFT] + [K]EXPORTAR [CTRL] + [SHIFT] + [E]

4. Guia de Estilos

Através de suas funcionalidades o Figma permite a ágil perfomance de guias de estilos, permitindo um maior controle organizacional de elementos bases de um projeto.

4.1 Styles

Os estilos são pré-definições globais de propriedades de elementos. Podemos utilizar como principais exemplos: paleta de cores, estilos de texto, etc.

4.2 Components [C]

São composições de elementos, facilitam a organização, agilidade e reutilização de elementos, através de sua hierarquia que mantém características globais dentro de um arquivo. Ex: Botões, Cabeçalhos, Rodapés, etc.

4.3 Nesting [C]

O nesting ou aninhamento, permite a criação de categorias de elementos, componentes e propriedades. Através da nomenclatura, utilizando o caractére '/' você pode construir uma categoria.
Ex¹: Cores Primárias / Cor 1.
Ex²: Cores Secundárias / Cor 1.

5. Prototipação

Um dos pontos fortes do Figma é sua funcionalidade com protótipos reais. Através de seus modos é possível o desenvolvimento de interfaces e projetos interativos e funcionais.

5.1 Modos de Design

O Figma possui três modos de operação:
- O modo de Design (padrão), onde você define os elementos gráficos;
- O modo Prototype, permite a interatividade dos elementos gráficos;
- O modo Code, compartilha as informações em código (CSS) dos elementos gráficos.

5.2 Prototipando

Utilizando o modo Prototype vamos apresentar algumas funcionalidades para o desenvolvimento de um protótipo funcional.

5.3 Figma Mirror

O Figma Mirror é um aplicativo que você instala no computador e entra com a mesma conta na qual está acessando o projeto e funciona em conjunto com o programa nos dispositivos. Você precisar selecionar um quadro no computador pra ele ser carregado no Mirror e a partir daí você pode interagir normalmente.
